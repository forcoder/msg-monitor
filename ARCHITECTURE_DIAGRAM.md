## 3. 模块依赖关系

### 3.1 Gradle模块结构
```
:app (主模块)
├── :core:ui (UI组件库)
├── :core:data (数据层)
├── :core:domain (领域层)
├── :feature:monitor (消息监控功能)
├── :feature:reply (回复生成功能)
├── :feature:knowledge (知识库功能)
├── :feature:model (模型管理功能)
└── :feature:style (风格学习功能)
```

### 3.2 依赖关系矩阵
```
模块                 依赖的模块
────────────────────────────────────────────────────
:app                所有功能模块 + core模块
:feature:monitor    :core:data, :core:domain
:feature:reply      :core:data, :feature:knowledge, :feature:model
:feature:knowledge  :core:data, :core:domain
:feature:model      :core:data, :core:domain
:feature:style      :core:data, :feature:reply
:core:ui            :core:domain
:core:data          :core:domain
:core:domain        无（纯Kotlin）
```

┌─────────┬─────────────────────────────────────────────┐

├─────────┼─────────────────────────────────────────────┤
│ VERBOSE │ 详细调试信息，开发阶段使用                   │

## 6. 安全架构

### 6.1 安全层次
```
应用层安全
├── 权限管理（运行时权限申请）
├── 数据加密（Keystore + Room加密）
├── 代码混淆（ProGuard/R8）
└── 反调试保护

传输层安全
├── TLS 1.3加密通信
├── 证书固定（Certificate Pinning）
├── 请求签名验证
└── 防重放攻击

API层安全
├── API密钥轮换机制
├── 访问频率限制
├── 异常行为检测
└── 数据脱敏处理
```

### 6.2 隐私保护
```
用户数据 → 本地加密存储 → 去标识化处理 → AI API调用
    │           │               │              │
    │           │               └──────────────┘
    │           │               仅发送必要信息
    │           │
    │           └─────────────────┘
    │            不存储原始消息内容
    │
    └─────────────────────────────────┘
        用户可随时删除个人数据
```
