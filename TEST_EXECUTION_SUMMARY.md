# 测试执行摘要

## 应用概述
csBaby 客户服务中心助手是一个 Android 应用程序，具有以下核心功能：
- **消息监听服务**：监控选定应用的实时通知
- **AI 回复生成**：多模型支持的智能回复系统
- **知识库管理**：客户服务的知识条目管理
- **悬浮窗支持**：提供覆盖层功能
- **无障碍自动化**：聊天交互自动化

## 测试优先级矩阵

| 测试类型 | 优先级 | 关键功能 |
|---------|--------|----------|
| 安全测试 | 紧急 | 权限管理、数据加密、输入验证 |
| 功能测试 | 高 | 消息监听、AI回复、知识库管理 |
| UI测试 | 中等 | 用户界面、主题切换、导航 |

## 测试覆盖率目标

- **单元测试**: ≥80% 代码覆盖率
- **集成测试**: 所有主要用户流程
- **安全测试**: 100% 关键安全功能覆盖
- **UI测试**: 所有主要屏幕和交互

## 执行环境要求

- **操作系统**: Linux/macOS/Windows (通过 WSL)
- **Java版本**: 17+
- **Gradle版本**: 7.0+
- **Android SDK**: API 26+
- **测试设备**: Android 模拟器或物理设备

## 快速开始

```bash
# 授予执行权限
chmod +x run-tests.sh

# 运行所有测试
./run-tests.sh

# 仅运行安全测试
./run-tests.sh --security

# 生成覆盖率报告
./run-tests.sh --report
```
